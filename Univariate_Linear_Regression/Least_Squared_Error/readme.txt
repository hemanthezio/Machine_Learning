---------------------------------------------------------------
Simple linear regression using Least Squared Error method
---------------------------------------------------------------
y = b_0 + b_1 x
SS_xy = Sxy - (Sx*Sy) / n
SS_xx = Sx^2 - (Sx)^2 / n
b_1 = SS_xy / SS_xx 
b_0 = (Sy / n) - b_1 * (Sx/n)
---------------------------------------------------------------
Performance evaluation is done using R Squared method
---------------------------------------------------------------
RESULTS:
R SQUARED = 0.975909898938941
